# Description

Из банка стали уходить клиенты. Необходимо спрогнозировать, уйдёт клиент из банка в ближайшее время или нет.

Столбец *Exited* отражает факт ухода клиента.

### Метрики

* *F1*-score > 0,59;
* *ROC-AUC*.
